# Scripts

## Automatic Office Installer

This is a script that automatically install Microsoft Office 365 for Enterprise. The products included are only the necessities such as Word, PowerPoint, Excel, etc. and does not bloat your system with other products like Bing, Outlook, etc.

```pwsh
iwr -useb https://raw.githubusercontent.com/dentolos19/dentolos19/refs/heads/main/scripts/install-office.ps1 | iex
```
